## 1. GraphRAG in 60 Seconds

### Build Your First Knowledge Graph

```typescript
// 1. Extract entities from text
const result = await invoke('graphrag_build_graph', {
  text: `
    Alice is a software engineer at Microsoft.
    Microsoft develops Azure cloud platform.
    Bob works with Alice on Azure projects.
  `
});

// 2. Search the graph
const search = await invoke('graphrag_retrieve', {
  query: 'Who works on Azure?'
});

console.log(search.results);
// â†’ Returns Alice and Bob with relevance scores

// 3. View statistics
const stats = await invoke('graphrag_stats');
console.log(stats);
// â†’ { total_entities: 4, total_relationships: 3, ... }
```

### Try It Now

1. Open the **Knowledge Graph** panel
2. Paste some text about your project/team
3. Click "Build Graph"
4. Search for entities or relationships

## 2. ReAct Agent in 60 Seconds

### Solve Your First Problem

```typescript
// Execute a reasoning task
const result = await invoke('react_execute', {
  query: 'What is 15% of 350, and is that more than 50?'
});

console.log(result.steps);
// â†’ Shows reasoning steps:
// Thought: I need to calculate 15% of 350
// Action: calculator (350 * 0.15)
// Observation: Result is 52.5
// Thought: Now I need to compare 52.5 to 50
// Answer: 15% of 350 is 52.5, which is more than 50

console.log(result.final_answer);
// â†’ "15% of 350 is 52.5, which is more than 50"
```

### Try It Now

1. Open the **ReAct Agent** panel
2. Enter a task like:
   - "Search for the latest news on AI and summarize it"
   - "Calculate the area of a circle with radius 5"
   - "Find information about quantum computing"
3. Watch the reasoning steps unfold
4. Get your answer!

## 3. Plan-and-Solve in 60 Seconds

### Create Your First Plan

```typescript
// 1. Generate a plan
const plan = await invoke('planner_generate', {
  goal: 'Create a report on climate change with data and charts'
});

console.log(plan.steps);
// â†’ Shows 5-7 steps with dependencies and estimated time

// 2. Review and approve

await invoke('planner_approve', { planJson });

// 3. Execute the plan

  planId: plan.id
});

console.log(execution);
// â†’ Shows execution results, logs, and final outcome
```

1. Open the **Plan-and-Solve** panel

3. Review the generated plan

## ðŸŽ“ 5-Minute Tutorials

### Tutorial 1: Build a Personal Knowledge Base

**Goal:** Create a knowledge graph from your notes/documents

```typescript
// Step 1: Prepare your content
const myNotes = `
  Project Alpha started in January 2024.
  Led by Sarah Chen, a senior architect at TechCorp.
  Uses React and TypeScript for the frontend.
  Backend built with Rust and PostgreSQL.
  Team includes developers from Seattle and Tokyo offices.
`;

// Step 2: Build the graph
await invoke('graphrag_build_graph', { text: myNotes });

// Step 3: Explore connections
const sarah = await invoke('graphrag_search_entities', {
  query: 'Sarah Chen',
  limit: 1
});

const neighbors = await invoke('graphrag_get_neighbors', {
  entityId: sarah.entities[0].entity_id
});

console.log('Sarah is connected to:', neighbors);
// â†’ Shows Project Alpha, TechCorp, React, TypeScript, etc.
```

**Time:** 2 minutes

### Tutorial 3: Project Planning

**Goal:** Plan a software project

```typescript
// Generate a comprehensive plan
const plan = await invoke('planner_generate', {
  goal: `
    Create a new web app for task management.
    Include: database design, API endpoints, frontend components,
    authentication, and deployment strategy.
  `
});

// Review the plan
console.log('Steps:', plan.steps.length);
console.log('Estimated time:', plan.estimated_time);
console.log('Required tools:', plan.required_tools);
console.log('Risks:', plan.risks);

// Approve and execute
if (confirm('Approve this plan?')) {
  const planJson = JSON.stringify(plan);
  await invoke('planner_approve', { planJson });
  const execution = await invoke('planner_execute', {
    planId: plan.id
  });

  console.log('Completed:', execution.completed_steps);
  console.log('Results:', execution.final_result);
}
```

**Time:** 5 minutes

## ðŸ”¥ Power User Tips

### Tip 1: Combine Features

Use GraphRAG + ReAct together:

```typescript
// 1. Build knowledge from research
const research = await invoke('react_execute', {
  query: 'Research top 5 programming languages in 2024'
});

// 2. Extract to knowledge graph
await invoke('graphrag_build_graph', {
  text: research.final_answer || ''
});

// 3. Query the graph
const languages = await invoke('graphrag_search_entities', {
  query: 'programming language',
  limit: 10
});
```

### Tip 2: Batch Processing

Process multiple documents:

```typescript
const documents = [
  'document1.txt',
  'document2.txt',
  'document3.txt'
];

for (const doc of documents) {
  const content = await readFile(doc);
  await invoke('graphrag_build_graph', { text: content });
}

// Now you have a unified knowledge graph!
```

    const subPlan = await invoke('planner_generate', {
      goal: step.description

    // Execute sub-plan...
  }

## ðŸ†˜ Troubleshooting

### Issue: "Ollama API call failed"

**Solution:** Ensure Ollama is running
```bash
ollama serve  # Start Ollama if not running
curl http://localhost:11434/api/tags  # Verify it's up
```

### Issue: "No entities found"

**Solution:** Your text may not contain recognizable entities. Try:
- Use proper names (people, organizations, places)
- Include specific technologies, projects, or concepts
- Provide more context

### Issue: "Plan execution failed"

**Solution:**
- Check execution logs for specific errors
- Ensure required tools are available
- Simplify the goal or break into smaller steps
