### 1. GraphRAG - Knowledge Graph-based Retrieval

Transform your conversations and documents into an intelligent knowledge graph.

**Key Features:**
- **9 Entity Types**: Person, Organization, Location, Technology, Concept, Tool, Project, Document, Event
- **8 Relationship Types**: WorksWith, PartOf, Uses, Creates, Knows, LocatedAt, DependsOn, RelatesTo
- **Community Detection**: Automatically groups related entities using label propagation
- **Multi-hop Retrieval**: Find information through relationship chains (up to 2 hops)
- **SQLite Storage**: Persistent knowledge graphs with full CRUD operations

**Use Cases:**
- Personal knowledge management
- Team collaboration tracking
- Project documentation
- Research organization
- Meeting notes analysis

## ðŸ”„ Migration Guide

### From v3.6.0 to v3.7.0

**No Breaking Changes!** v3.7.0 is fully backward compatible.

**New Dependencies:**
- None (all features use existing dependencies)

**Database Changes:**
- New tables: `kg_entities`, `kg_relationships`, `kg_entity_documents`, `kg_communities`
- Existing data is not affected

**Configuration:**
- New config files: None (configs are in-memory by default)
- Existing settings: All preserved

**Steps to Upgrade:**
1. Pull latest code: `git pull origin main`
2. Install dependencies: `npm install && cd src-tauri && cargo build`
3. Run the app: `npm run dev`
4. Explore new features in the UI!
